How this map works

● Verified — merged into main, visible to everyone.

● Pending — proposed in an open pull request, not yet reviewed. Only shown when you switch to "Verified + Pending".

Click + Propose marker, then click a spot on the map. You'll fill in a short form and get a one-click link that opens GitHub with your marker file pre-filled — just review it and click "Propose new file" to open a pull request.

Owner mode

Connect your GitHub token

Paste a fine-grained personal access token scoped only to this repo, with Contents: Read and write permission. This lets you attach photos directly (uploaded straight into the repo) and publish your own markers to main instantly, skipping the pull request step other people go through.

The token is stored only in this browser's local storage. It is never sent anywhere except directly to api.github.com from your own device. Don't paste it on a shared computer, and don't share this token with anyone.
